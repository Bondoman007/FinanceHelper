- Income vs. expense comparison

- Interactive Pie Charts (category breakdown)

- Session management

**Frontend:**
- React 18 + Vite
- Tailwind CSS
- Recharts
- React Router v6

**Backend:**
- Node.js/Express

- JWT Authentication
